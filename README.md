# Askida Yemek

Agentic AI Destekli Gida Israfi Onleme Uygulamasi

## Ekip (7 Kisi)
- Frontend (2 kisi) - React Native / Expo
- Backend (3 kisi) - Python / FastAPI
- AI (2 kisi) - Groq / Llama 3

## Kurulum
### Frontend
```bash
cd frontend
npm install
npx expo start
```
### Backend
```bash
cd backend
pip install -r requirements.txt
```
### AI
```bash
cd ai
pip install -r requirements.txt
```
